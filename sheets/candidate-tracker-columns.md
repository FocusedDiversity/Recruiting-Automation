# Candidate Tracker — Column Spec

Create one Google Sheet per role. Use the following columns:

| Column | Description |
|---|---|
| Timestamp | Auto-filled by Make when application received |
| Full Name | Candidate full name |
| Email | Candidate email |
| Phone | Candidate phone number |
| LinkedIn URL | Link to LinkedIn profile |
| Resume Link | Google Drive link to resume file |
| Cover Letter Link | Google Drive link to cover letter file |
| Candidate Folder Link | Google Drive link to candidate folder |
| LLM Score | Overall score (1-10) from LLM evaluation |
| LLM Summary | 2-3 sentence fit summary |
| Strengths | Bulleted strengths from LLM |
| Concerns | Bulleted concerns/gaps from LLM |
| Synaptitudes Alignment | Notes on cultural fit |
| Recommended Questions | Interview questions from LLM |
| LLM Decision | Advance / Hold / Pass |
| Stage | Current interview stage (updated manually) |
| Hiring Manager Notes | Free text notes |
